# JS V1 - Day 19 - 23

## Task
Ek aisa project banaiye jisme **Admin Panel** ho. Isme complete **Navbar, Sidebar aur Forms** hone chahiye.  

### Sidebar Functionality
- Navbar me ek **Menu Button** hona chahiye.
- Menu button par click karne par **Sidebar open aur close** hona chahiye.

### Forms Show/Hide
Sidebar ke andar 2 options hone chahiye:

- **Signup**
- **Login**

User jab **Signup** par click kare toh **Signup Form** show ho.  
User jab **Login** par click kare toh **Login Form** show ho.  

⚡ Dono forms ko **JavaScript ke through dynamically show/hide** karna hai.

---

## Form Validation
Dono forms me proper validation lagani hai:

### Signup Form Validation
- Name empty na ho
- Email valid format me ho
- Password minimum length check ho
- Confirm Password match kare
- Error hone par proper **error message display** ho

### Login Form Validation
- Email valid ho
- Password empty na ho
- Wrong credentials par error message show ho
