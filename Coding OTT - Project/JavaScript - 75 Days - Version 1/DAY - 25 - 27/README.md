# JS V1 - Day 25 - 27

## Task
Ek aisa project banaiye jisme user ek **Text Message** enter kare aur saath hi ek **Repeat Number** enter kare.

---

## Working
- User jo message dalega, usko **loop** ki help se repeat karna hai.
- Repeat count user ke input ke according hona chahiye.

Example:  
Message: `Hello`  
Repeat: `3`  

Output:  
Hello  
Hello  
Hello  

---

## Min / Max Validation
Repeat number ke liye **minimum aur maximum limit set** karni hai.

✅ Example:  
- Minimum: `1`  
- Maximum: `10`

Agar user minimum se kam ya maximum se zyada number enter kare toh:

- **Error message show** karna hai
- Output generate nahi hona chahiye

---

## Output & Error Display
- **Error message** ko text me show karna hai.
- **Repeated output message** ko bhi text format me show karna hai.

---

## Important Instruction
Project ki saari functionality **JavaScript** aur **loops** ka use karke implement karni hai.
