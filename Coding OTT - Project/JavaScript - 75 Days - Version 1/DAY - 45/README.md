## JS - V1 - DAY: 45

### 📌 Task:
Ek **professional navbar** design karo jisme:

- **Left side** me ek **icon/logo**, website ka **name**, aur ek **tagline** ho.
- **Right side** me proper **menu links** ho.
- Navbar ko **fully responsive** banana hai (mobile + tablet + desktop).

---

### ✅ Navbar Requirements:

- Left side:
  - Website icon/logo
  - Website name
  - Tagline (small text)

- Right side:
  - Menu links (Home, About, Courses, Contact etc.)
  - Mobile view me **hamburger menu** show hona chahiye
  - Menu open/close properly work kare

---

### 📚 Course Cards Section:

Navbar ke niche ek **Courses Section** banana hai jisme multiple **course cards** ho.

Har course card me:

- **Course Name**
- **Course Price**
- **Purchase Button**

---

### 🎨 Design Requirements:

- Cards ka UI **modern aur professional** hona chahiye.
- Proper spacing, padding, hover effects, shadows, border radius, etc. use karo.
- Cards ko responsive grid layout me arrange karo:
  - Mobile: 1 card per row
  - Tablet: 2 cards per row
  - Desktop: 3-4 cards per row
