# To-Do List App (Beginner + Powerful)

## 🔥 Banana kya hai?

### Ek simple To-Do app jisme user:

- task add kare
- task delete kare
- task complete mark kare
- tasks list screen pe show ho

### Features:
- Input box + Add button
- List me tasks show
- Delete button har task ke saath
- Complete button (task pe line-through)

### Topics Used:
- ✅ Variables
- ✅ Functions + Parameters
- ✅ Array (push, splice)
- ✅ Loops (tasks render karne ke liye)
- ✅ Conditionals (empty input check)
- ✅ DOM selectors + .innerHTML
- ✅ Boolean (completed true/false)

### Extra Improvement:
- Agar input empty ho to error message show karna.
- Total tasks count show karna.
