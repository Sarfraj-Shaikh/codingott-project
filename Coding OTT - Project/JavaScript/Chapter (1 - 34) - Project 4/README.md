# Quiz App (MCQ System)

## 🔥 Banana kya hai?
Ek **Quiz App** banana hai jisme **5–10 questions** hon.

## ✅ Requirements
- Har question ke **4 options** hon: **A, B, C, D**
- User option select kare
- Phir **Next Question** pe jaye
- Last me **final score show** ho

---

## 📌 Structure (Array of Objects Format)
Questions ko **Array of Objects** me store karna hai:

```js
[
  {
    question: "JS ka full form?",
    options: ["JavaSource", "JavaScript", "JustScript", "None"],
    correct: 1
  }
]

---

## ⚙️ Features

- Question display

- Options buttons (A, B, C, D)

- Next button

- Score counter

- End screen result

---

## 🧠 Topics Used

- ✅ Arrays of Objects
- ✅ Loops
- ✅ Functions
- ✅ Conditionals
- ✅ Boolean (correct/wrong)
- ✅ `.innerHTML`
- ✅ Switch case (optional)
- ✅ Ternary operator (optional)

---

## 🚀 Extra Improvement

Har **correct answer** pe option **green highlight**

Har **wrong answer** pe option **red highlight**
