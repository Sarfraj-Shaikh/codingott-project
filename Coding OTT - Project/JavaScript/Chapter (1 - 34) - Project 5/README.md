# 🎯 Number Guessing Game

## 🔥 Banana kya hai?
Computer ek **random number (1-50)** generate karega.  
User ek number guess karega, aur system batayega:

- **Too High**
- **Too Low**
- **Correct**

User ke **attempts count** honge.  
Correct hone par final message:

✅ **"Correct! You took 6 attempts."**

---

## ⭐ Features
- 🎲 Random number generate (1-50)
- 🔢 Attempts count
- 🔁 Reset button
- 💡 Hint system:  
  Agar difference **10 se kam** ho to show karo: **"Very Close"**
