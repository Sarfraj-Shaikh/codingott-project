## Chapter 12 – Basic Web App 3

## Task 1:
Develop a web application that displays an image along with two buttons.

When the first button is clicked, the height and width of the image should change dynamically.

When the second button is clicked, a border should be added to the image.

## Task 2:
Develop a web application that includes the following components:

An <h2> element displaying a question (e.g., What is JavaScript?)

Multiple buttons representing answer options (e.g., Programming Language, English Language)

A <textarea> element to display the result

When a user clicks on any option button, the application should display “Right Answer” or “Wrong Answer” in the textarea based on the selected option.

## Task 3:
Develop a web application that contains:

• An input field
• A button

When the button is clicked, the value entered in the input field should be displayed on the screen.
