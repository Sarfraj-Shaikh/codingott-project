# Chapter 13: Address Maker

Please complete the following tasks according to the instructions given below.

---

## Task 1

Create **two input fields** and add a **Show** button.  
When the user clicks on the button, the **first name** and **last name** entered by the user should be combined and displayed together in the output section.

---

## Task 2

Create a form where the user can enter the following details:

- Name  
- Email  
- Phone Number  
- Relation Type (S/O, W/O, C/O)  
- City  
- State  
- Country  
- Full Address  

After the user fills in all the required information, display the **complete and properly formatted address** in the output section below the form.

---

## Customization Options

Provide the following customization features for the displayed address:

### Text Style
- Bold  
- Italic  
- Underline  

### Text Alignment
- Left  
- Center  
- Right  

### Text Color
- Red  
- Blue  
- Green  
- Black  

---

## Print Options

Include a **Print** feature so the user can print the address whenever required.

### Conditions

There should be **two print buttons**:

1. **Print Full Page**  
   - Clicking this button should print the complete page.

2. **Print Address Only**  
   - Clicking this button should print **only the address**, while all other elements on the page should be hidden during printing.
