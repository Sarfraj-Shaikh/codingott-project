# Chapter 15: Range Validator

## Task:
Kripya ek project develop karein jisme ek text aur ek progress bar (range input) ho.  

Is progress bar ke liye minimum aur maximum values set honi chahiye.

Jab user progress bar ko manually adjust kare, to uske according text ka font size dynamically change hona chahiye.  

Saath hi, screen par yeh message clearly display hona chahiye:

**"Your font size is: __ px"**

Is project ka objective yeh hai ke range input aur text styling ke beech proper interaction demonstrate kiya ja sake.
