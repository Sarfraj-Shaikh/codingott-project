# Chapter 17: Image Effects

## Task
Kripya ek aisa project develop kijiye jisme ek image aur progress bars shamil ho.  

Har progress bar ki **minimum** aur **maximum** values pehle se defined honi chahiye.

Progress bars ko adjust karne par image ki **brightness**, **contrast**, aur **saturation** ko control kiya ja sake.

In teeno effects ke liye alag-alag **range inputs** hone chahiye, jisse user inhe manually adjust kar sake.

## Conditions
- Project mein **webkit** ka istemal karna anivarya hai aur sahi **properties** ke saath use kiya jaana chahiye.
