# Chapter 18: Comparison Operator

## Task 1
Ek input field banani hai jisme user khud se koi value aur operator enter kare.
  
Jab user **Show** button par click kare, to uska result screen par display ho.

## Task 2
Teen input fields banani hain:

- Pehli input me user number enter kare  
- Beech wali input me user operator ka sign enter kare  (`*`, `+`, `-`, `%`, `/`, `<`, `>`, `<=`, `>=`, `==`, `===` etc.)
- Teesri input me user number enter kare  

Jab user **Calculate** button par click kare, to calculation ka result show ho.

## Condition
Is project ko banate waqt `eval()` function ka istemal karna **mandatory** hai.
