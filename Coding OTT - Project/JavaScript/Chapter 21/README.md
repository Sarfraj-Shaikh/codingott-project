# Chapter 21: Form Validation and Submission

## Task

Kripya ek aisa form banayein jisme user se **Name** aur **Email ID** li ja rahi ho.

Form submit hone par yeh verify kiya jaye ki user ne wahi values enter ki hain jo pehle se define ki gayi hain.

- Agar entered values expected values se match karti hain, to **“Success”** ka message display ho.

- Agar entered values match nahi karti hain, to **“Failed”** ka message display ho.

Is task ko complete karte waqt **Logical Operators** ke saath **Ternary Operator** ka istemal karna **required** hai.
