## Chapter 24: Part 1

### Task 1:
Kripya ek aisa project tayar kijiye jisme:
- Ek **button** ho.
- Button par click karne par **JavaScript function** call ho.
- Function ka **result `<p>` tag** ke andar display ho.

**Conditions:**
- `onclick` event **sirf JavaScript code** ke through handle kiya jaye.
- HTML ke andar `onclick` attribute ka **bilkul istemal na ho**.
- HTML file me sirf **button** aur uski **id** di jaye.

### Task 2:
Kripya ek aisa project banaiye jisme:
- Button par click karne par **`setAttribute()` function** call ho.
- Is function ke zariye **text** aur **div** elements ko required **attributes** assign kiye ja saken.
