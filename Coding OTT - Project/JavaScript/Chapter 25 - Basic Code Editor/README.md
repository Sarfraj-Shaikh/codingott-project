# Chapter 25: Basic Code Editor

## Task:

Kripya ek aisa project develop kijiye jisme heading ke roop mein **"BASIC CODE EDITOR"** likha ho aur uske saath version bhi clearly mention ho.

Iske baad, code likhne aur uska output dikhane ke liye **fieldset** tag ka upyog kijiye. Fieldset ke andar **legend** tag ka use karke ek appropriate heading dijiye. Isi fieldset ke andar **textarea** ka istemal kijiye, jisme user apna HTML code likh sake.

fieldset ke neeche ek **"Run Code"** button hona chahiye. Button par click karne par, likha gaya code execute ho aur uska **output dusre fieldset me clearly display** ho.
