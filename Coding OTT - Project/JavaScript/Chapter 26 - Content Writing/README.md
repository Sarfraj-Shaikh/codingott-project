## Chapter 26: Content Writing

### Task:
Please ek aisa project create karo jisme following JavaScript functions ka use ho:

- `createElement()`
- `createTextNode()`
- `appendChild()`

In functions ki help se ek HTML tag create karo, uske andar value enter karo, aur phir us created tag ko document ke body me add karo.

## Suggested Tasks

### 1. Dynamic Heading Create Karna  
JavaScript se ek `<h1>` tag create karo, usme text add karo  
(jaise **“Welcome to My Website”**) aur usko body me insert karo.

### 2. Paragraph Generator  
Button click par ek naya `<p>` tag create ho aur usme user-defined text show ho.

### 3. List Banana (UL / OL)  
JavaScript ke through `<ul>` aur `<li>` tags create karke ek dynamic list display karo.

### 4. Profile Card  
Ek `<div>` ke andar `<h2>` aur `<p>` tags create karo jisme name aur description show ho.

### 5. Notification Message  
Ek `<div>` ya `<span>` tag create karke success ya warning message body me add karo.

### 6. Footer Text Add Karna  
Page load hote hi JavaScript se footer text create ho aur page ke bottom me add ho.

### 7. Button Create Karna  
JavaScript se `<button>` tag create karo aur usme text add karke body me show karo.
