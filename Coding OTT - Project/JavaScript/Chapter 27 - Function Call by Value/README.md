## Chapter 27: Function Call by Value

### Task:
Kripya ek aisa project banayein jisme *do input fields* hon aur ek *Calculate* button ho.  
Jab *Calculate* button par click kiya jaye, to *main function* call ho.

Main function ke andar *addition (+), subtraction (-), multiplication (), division (/), aur modulus (%)* ke liye **alag-alag functions* call hone chahiye.

Har operation ke liye *separate function* hona zaroori hai.  
Main function se jab in functions ko call kiya jaye, to input data *parameters ke through pass* kiya jaye.

Baaki sabhi functions ko parameters ke zariye data *receive karke apna respective operation perform* karna hoga.
