# Chapter 30: Part 1 - Image Hide & Show

## Task:
Ek aisa project banaiye jisme ek **card** ho. Card ke andar ek **image** ho aur image ke niche ek **button** ho. Button par click karne par image **hide** ho jaye aur dobara click karne par image **show** ho jaye.

## Conditions:
Image ko **hide & show** button ke text ke according control karna hai.

- Agar button par text ho **"Show Image"**, toh click karne par image **show** honi chahiye aur button ka text change hokar **"Hide Image"** ho jana chahiye.
- Agar button par text ho **"Hide Image"**, toh click karne par image **hide** honi chahiye aur button ka text change hokar **"Show Image"** ho jana chahiye.
