# Chapter 30 - Part 2 (Main Projects)

## Project 1: Grade Checker Project
Ek project banao jisme user input field me marks enter kare aur button click karne ke baad `<p>` tag me grade show ho.

### Conditions:
- Agar marks >= 80 ho to show karo: **Grade A**
- Agar marks >= 60 ho to show karo: **Grade B**
- Agar marks >= 40 ho to show karo: **Grade C**
- Otherwise show karo: **Fail**

### Rules:
- if-else ka use karna zaroori hai.
- Output `<p>` tag me show hona chahiye.

## Project 2: Login Checker Project
Ek project banao jisme user username aur password enter kare aur login button par click kare.

### Conditions:
- Agar username == "admin" aur password == "1234" ho to show karo: **Login Successful**
- Otherwise show karo: **Invalid Username or Password**

### Rules:
- if-else ka use zaroori hai.
- `&&` operator ka use karna hai.
- Output `<p>` tag me show hona chahiye.

## Project 3: Discount Calculator Project
Ek project banao jisme user bill amount enter kare aur button click karne ke baad discount message show ho.

### Conditions:
