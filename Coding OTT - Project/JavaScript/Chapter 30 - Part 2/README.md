# Chapter 30 - Part 2

## Task 1: Even / Odd Checker Project
Ek aisa project banao jisme user input field me number enter kare aur button par click karne ke baad `<p>` tag me result show ho ki number **Even** hai ya **Odd**.

### Conditions:
- Even/Odd check karne ke liye **if-else** condition ka use karna zaroori hai.

## Task 2: Zoom In / Zoom Out / Reset Project
Ek aisa project banao jisme ek text ho aur uske niche 3 buttons hon:

1. **Zoom In**
2. **Zoom Out**
3. **Reset**

User jab buttons par click kare to text ka size increase/decrease ho.

Agar text size **maximum limit** ya **minimum limit** tak pahunch jaye to `<p>` tag me message show karo.

### Conditions:
- Saare actions **JavaScript onclick** se perform karne hain.
- User same button ko baar-baar click kar sake.
- Jab text **maximum size** tak pahunch jaye to message show karo.
- Jab text **minimum size** tak pahunch jaye to bhi message show karo.
