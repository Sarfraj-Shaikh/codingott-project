# Chapter 30 - Part 3

## Task:
Ek aisa project banayein jo **Facebook Signup page** jaisa design ho. Is project me **if-else** ka use karke validation lagani hai.

## Validation Requirements:
- Jab user **input field ke andar click kare**, toh kuch bhi nahi hona chahiye.
- Lekin jab user **input field me click karne ke baad bahar kahin aur click kare**, toh function call hona chahiye.
- Function input ko validate kare aur check kare ke user ne value enter ki hai ya nahi.
- Agar input field **empty ho**, toh:
  - Us input ka **border red** ho jana chahiye
  - Aur input box ke andar **red color ka warning icon** show hona chahiye
