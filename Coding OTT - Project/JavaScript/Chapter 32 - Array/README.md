# JavaScript Array Practice Tasks (Beginner Level)

## Rules (Must Follow)
For every task, always print:

- Before Array
- Method Used
- After Array

Example (JS):
Before: [1, 2, 3]
Method: push(4)
After: [1, 2, 3, 4]

---

# Task 1: Shopping List Manager

## Goal:
Shopping list ko manage karna using array methods.

## Tasks:
1. Ek array banao: ["Milk", "Bread", "Sugar"]
2. End me "Rice" add karo using push()
3. Last item remove karo using pop()
4. Start me "Tea" add karo using unshift()
5. First item remove karo using shift()
6. Beech me se "Bread" delete karo using splice()
7. Final list ko string me print karo using join(" | ")
