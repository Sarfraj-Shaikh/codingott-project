# 🥇 Project 1: Grocery / Inventory Manager

## 🎯 Goal:
Ek grocery inventory system banana using array methods.

## 📌 Tasks:
1. Ek array banao: `["Milk", "Bread", "Sugar"]`

2. End me `"Rice"` add karo using `push()`

3. Last item remove karo using `pop()`

4. Start me `"Tea"` add karo using `unshift()`

5. First item remove karo using `shift()`

6. `"Bread"` ko delete karo using `splice()`

7. Ek new delivery list banao: `["Oil", "Salt"]`

8. Dono arrays merge karo using `concat()`

9. Final grocery list ko string me print karo using `join(" | ")`

## ✅ Output Rules:
Har step ke baad ye print hona chahiye:

- Before Array  
- Method Used  
- After Array  

Aur last me:
