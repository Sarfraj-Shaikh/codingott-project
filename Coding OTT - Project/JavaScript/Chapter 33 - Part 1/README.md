# Chapter 34 - Part 1

## Task 1

Please ek aisa project banao jisme ek **array** ho jisme kuch persons ke **names** stored hon.  
Phir **JavaScript loop** ki help se un names ko print karo.

### Requirements:
- Loop utni baar chale jitne names array ke andar available hain.
- Output ko **text form** me show karna hai (screen par display).

## Task 2

Please ek aisa project banao jisme user jo bhi number enter kare, wo us number ka **table** find kar sake.

### Requirements:
- JavaScript me loop use karo.
- Loop ko **10 times** run karna hai.
- User ke entered number ko **1 se 10 tak multiply** karke table generate karna hai.
