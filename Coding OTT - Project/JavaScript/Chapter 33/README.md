# Chapter 33 (Loop)

## Task

Please ek aisa project banao jisme **2 inputs** hon:

1. **Message Input:**  
   User is input me apna message likhega.

2. **Number Input:**  
   User is input me number dalega jiske according **loop chalega**.

## Requirements

- User ne jitna number enter kiya ho, loop utni baar run hona chahiye.
- Message ko utni baar repeat karna hai jitni baar loop chale.
- Har repeated message ko **separately** show karna hai.
- Output **cards** ke form me hona chahiye.
- Har card ke andar **textarea** me message show hona chahiye.
- Sare cards **niche one-by-one** display hone chahiye.
