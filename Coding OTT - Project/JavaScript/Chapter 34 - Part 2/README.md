# Chapter 34 - Part 2

## Task 1 (Even Numbers)

Please ek aisa project banao jisme user se **ek number input** liya jaye.  
Phir JavaScript me **loop** chala kar check karo ke user ka number **even** hai ya nahi.

### Requirements:
- Agar number **even** ho, toh us number se pehle ke **saare even numbers** print karo.
- Output ko **text form** me screen par show karna hai.

## Task 2 (Odd Numbers)

Same project odd number ke liye banao.

### Requirements:
- User se ek number input lo.
- JavaScript loop ki help se check karo ke number **odd** hai ya nahi.
- Agar number **odd** ho, toh us number se pehle ke **saare odd numbers** print karo.
- Output ko **text form** me screen par show karna hai.
