# Chapter 35

# JavaScript DOM Practice Tasks

## 1. Button Click Alert
**Task:**  
- Ek button create karein.  
- Button click karne par alert message show ho.  

**Focus:**  
- `addEventListener()`  
- function call practice  

---

## 2. Change Text on Click
**Task:**  
- Ek paragraph ya heading create karein.  
- Button click par text change ho jaye.  

**Extra:**  
- Ek aur button add karein jo text ko original state me wapas le aaye.  

---

## 3. Simple Counter
**Task:**  
- Ek button aur ek number display create karein.  
- Har click par number `+1` increase ho.  

**Extra:**  
- Ek reset button add karein jo counter ko `0` kar de.  

---

## 4. Show/Hide Element
**Task:**  
- Ek paragraph aur ek button create karein.  
- Button click par paragraph show/hide ho jaye.  

**Focus:**  
- `element.style.display`  
- click events  

---

## 5. Color Changer
**Task:**  
- Ek div create karein jiska background color button click par change ho.  

**Extra:**  
- Multiple buttons add karein different colors ke liye.
