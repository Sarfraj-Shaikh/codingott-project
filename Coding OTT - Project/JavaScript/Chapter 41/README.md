# Chapter 41

## Task

1. Ek project banao jisme browser ke **URL ko secure** kiya jaaye using `history.pushState()`.  
   - Ensure karo ki URL update ho lekin page reload na ho.  
   - Agar user **page refresh** kare, koi error na aaye.  

2. Page par **ek separate refresh button** add karo jo user ko manually page reload karne ka option de.  

3. **Right-click disable** karo taaki context menu na aaye.  

4. **Copy option disable** karo aur keyboard se copy (Ctrl+C) rokne ke liye `onKeyDown` event ka use karo.  
   - `event.ctrlKey` aur `event.key` ya `event.keyCode` ka condition check karke copy ko prevent karo.  

5. Ensure karo ki ye features **har browser action ke liye consistent** aur smooth kaam kare.
