## Task

Ek aisa project banao jisme **4 input fields** hon aur unme **Credit/Debit Card Number** enter kiya jaye.

### Requirements:
- Total **4 inputs** hon.
- Har input field me user sirf **maximum 4 digits** hi type kar sake.
- Jaise hi user **4 digits complete** kare, cursor **automatically next input** me chala jaye.

### Example:
`1234 5678 9012 3456`
