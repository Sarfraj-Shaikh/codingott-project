# Chapter 45

## Task

Ek aisa project banao jisme user **theme change** kar sake (Light Mode / Dark Mode) aur user ki selected theme **Cookies me store** ho jaye.

### Requirements:
- User ko **theme switch** karne ka option mile.
- User jo bhi mode select kare (Light/Dark), wo **cookie me save** ho.
- Page refresh hone ke baad bhi user ki **last selected theme automatically apply** ho jaye.

### Example:
Agar user ne **Dark Mode** select kiya hai, toh next time page open karne par bhi **Dark Mode hi active** ho.
