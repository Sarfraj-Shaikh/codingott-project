# Chapter 46

## Task

Ek aisa project develop karna hai jisme **username aur password ke through login** ki facility ho.

Login form ke niche:

- **Left side** par **"Remember Me"** ka checkbox ho  
- **Right side** par **Login** ka button ho  

Agar user **"Remember Me"** checkbox select karta hai, toh **username aur password cookies me store** ho jana chahiye.

Jab bhi user dobara login page open kare, toh **cookies se data automatically load** ho kar fields me show ho.

Agar user **"Remember Me"** ko uncheck kar deta hai, toh **cookies se stored data remove** ho jana chahiye.
