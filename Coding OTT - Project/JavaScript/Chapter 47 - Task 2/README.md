## Task 2

Ek aisa project develop karna hai jisme jab user **website first time open** kare, toh uske samne ek **popup** show ho jisme user se **mobile number enter** karwaya jaye.

Jab tak user **mobile number enter karke submit** nahi karega, tab tak popup **hide nahi hona chahiye**.

Submit hone ke baad mobile number ko **cookies me store** karna hai.

Agar user dobara website open kare, toh **popup again show nahi hona chahiye**.
