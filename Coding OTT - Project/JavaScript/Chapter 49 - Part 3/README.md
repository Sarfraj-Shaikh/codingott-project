# Chapter 49 - Part 3

## Task:
Ek aisa project banao jisme **Flipkart jaisa Product View Section** ho.

### Features:
- Product ka **Name** aur **Price** show ho.
- Niche **2 CTA Buttons** ho:
  - **Buy Now**
  - **Add to Cart**
- Product ka **Image** show ho.
- Image ke **left aur right side** me buttons ho image change karne ke liye.
- **JavaScript** ki help se `scrollBy()` function use karke image ko scroll karke change karna hai.
