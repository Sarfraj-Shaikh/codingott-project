# Chapter 50 - Part 4

## Task:
Ek aisa project banao jisme pehle check ho ki **user ke browser me cookies enabled hain ya nahi**.

### Flow:
- Agar cookies **enabled** hain to user ko ek **image show** karo.
- Image ke niche **2 options** do:

### Options:
#### 1) Don't Show Again (Left Side)
- Agar user is option par click kare to image future me dobara show nahi honi chahiye.
- Uske badle **Welcome Message** show hona chahiye.

#### 2) Remind Me Later (Right Side)
- Isme user ke paas ye options hone chahiye:
  - **1 Week**
  - **15 Days**
  - **1 Month**

### Condition:
- Agar user **Remind Me Later** me se koi option select karta hai, to image **sirf selected time complete hone ke baad hi** dobara show honi chahiye.
- Agar selected time complete nahi hua, to image ki jagah **Welcome Message** show hona chahiye.
