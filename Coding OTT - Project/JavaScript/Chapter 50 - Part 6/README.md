# Chapter 50 - Part 6

## Task:
Ek aisa project banao jisme **Local Storage** ka use karke **CRUD operations** perform ho rahe ho.

### Requirements:
- **Create**: Local Storage me data store karna.
- **Read**: Local Storage se data retrieve karke show karna.

### Output:
- Har CRUD operation ka result **text format me screen par display** hona chahiye.
