# Chapter 50 - Part 8

## Task:
Ek aisa project banao jisme check ho ki **user ne JavaScript enabled rakhi hai ya disable**.

### Conditions:
- Agar user ne **JavaScript disable** kar rakhi hai to ek message show karo:
  - **"Please JavaScript enable karein"**
- Agar user ne **JavaScript enabled** rakhi hai to bhi text me show karo:
  - **"JavaScript is Enabled"**
